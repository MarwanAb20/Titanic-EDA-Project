# Titanic EDA and Data Preprocessing

This project explores the Titanic dataset using Exploratory Data Analysis (EDA) and Data Preprocessing techniques.

## Steps
- Data loading
- Data exploration
- Missing value handling
- Feature encoding
- Feature scaling

## Tools
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

Dataset:
https://www.kaggle.com/datasets/yasserh/titanic-dataset
